### analyse_elus

Description des différents fichiers : 

    AUTHORS.md : donne les noms des auteurs de l'étude

    elus-conseillers-departementaux-cd.csv : dataset qui sert de base pour l'étude réalisée

    Analyse_dataset_elus_departementaux.ipynb : jupyter notebook contenant l'analyse réalisée

### Exécution du notebook 

Pour exécuter le notebook, il vous faudra télécharger le fichier elus-conseillers-departementaux-cd.csv et le placer dans un même dossier que le notebook Analyse_dataset_elus_departementaux.ipynb.  

Modules Python nécessaires pour ce notebook : pandas, matplotlib, datetime.

# Important

Pour que tout fonctionne correctement, il ne faut exécuter qu'une seule fois chaque cellule de code.
